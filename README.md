# Todo List App

"A simple browser-based task manager that saves data to local storage."

## Features

- **Feature 1:** Add and delete tasks.
- **Feature 2:** Data persists after refreshing the page.

## Built With

- **HTML5** - Structure
- **CSS** - Styling (Grid, buttons, colors)
- **JavaScript** - Functionality

## How to Run

1. Clone the repository:
   `git clone https://github.com/towfique-pranto/js-todo-list`
2. Open `index.html` in your browser.
